# Repo Polish Notes

A useful automation repo should communicate value in the first 10 seconds.

## Minimum polish

1. Clear tagline
2. Problem statement
3. 30-second quickstart
4. Real commands
5. Safety model
6. License
7. Changelog
8. Roadmap
